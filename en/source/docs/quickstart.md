These commands will download, install, generate OpooPress site and start a web server to preview.
```
~ $ mkdir myblog
~ $ cd myblog
~ $ wget http://www.opoopress.com/downloads/pom.xml
~ $ mvn op:preview
```
Open http://localhost:8080/ in your browser to preview.
