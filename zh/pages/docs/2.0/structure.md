默认（推荐）的目录结构如下

```
(.)
 |
 +- pom.xml
 |
 +- config.yml
 |
 +- assets/
 |
 +- pages/
 |
 +- posts/
 |
 +- plugins/
 |
 +- themes/
     |
     +- default/
     |    |
     |    +- theme.yml
     |    |
     |    +- ...
     |
     +- another-theme/
```
